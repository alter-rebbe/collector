Сопроводительный документ ("вид"), выданный Шнеуру Залману при выезде из Петербурга для предъявления в канцелярии Белорусского гражданского губернатора

Дата: 1801-03-28 Кто: Кому: ⎙ 335 1 332 2 № 36 3

Объявитель сего еврей Зальманъ Бору- ховичъ находившись здѣсь по некоторомуъ дѣлу нынѣ по ненадобности въ немъ. надобностямъ нынѣ по высочайшему Государя импе ратора повеленію живущему въ домѣ его состоящей въ Белоруской губерніи въ местечѣ ліознѣ съ тѣмъ, чтобы онъ по прибытіи въ витебскъ явился съ симъ видомъ у Белорускаго Гна Гражданскаго Губернатора Госпо дина Къ которому об нем писано. Тайнаго Совѣтника Тар- бѣева С.п.бургъ марта 28 дня 1801. года
