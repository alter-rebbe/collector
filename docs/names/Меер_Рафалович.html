 Меер Рафалович Мейер Рафалович Меіер Рафалович Мейер Рафалс Меер Рафаилович Рафаелович

раввин, руководитель хасидской общины Вильно; один из первых хасидов города (по версии Нохима Ицковича с 1772 г.; по версии самого Рафаловича – с 1788 г., по данным Булгакова с 1799 г.), в доме М.Р. до октября 1797 г. проводились общинные молитвы хасидов; со времени полной смены состава виленского кагала, в 1799 - 1800 гг. – старший кагальный; торговец алкогольными напитками; родственник Нохима Ицковича, тесть Зелмана Леизеровича; по словам Авигдора Хаймовича – податель ложных жалоб на городских старшин
